Implemented in `renderer.ts` + widget loop in `index.ts`:
- `setWidget("midi-monitor", lines)` updated every 2 seconds
- Shows play/stop icon + BPM header
- Compact step-sequencer lines per channel with emoji by type
- Drum channels show K/S/H/C/R icons, melodic show note names
- Configured channels shown first in order, then unconfigured
- Toggle with Ctrl+M shortcut
