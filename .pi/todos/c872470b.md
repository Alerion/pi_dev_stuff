Implemented in `midi_monitor.py` ChannelTracker class:
- MIDI clock tracking (24 PPQN → 16th note grid)
- Note quantization to nearest step
- Loop boundary detection on pattern restart
- Pattern comparison between loops for stability detection
- Per-step: note number, velocity, gate length
- Polyphonic support (multiple notes per step for drums)
- BPM detection from clock timing (96-clock moving average)
