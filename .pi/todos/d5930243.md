Implemented `midi_monitor.py` - background Python process with:
- Auto-detects BeatStep Pro MIDI ports
- Buffers notes per channel, tracks MIDI clock for step quantization
- Pattern detection via loop boundary tracking
- TCP socket server with JSON IPC protocol
- Commands: get_patterns, get_pattern, get_devices, get_status, set_ports, set_pattern_length, quit
- Tested: starts cleanly, IPC works, detects BSP ports
